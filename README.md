## 🎬 Animations

This project includes two CSS keyframe animations:

### 📥 Drop Animation

The popup enters the screen with a bouncing drop effect.

```css
@keyframes drop;
```

### 🎯 Swing Animation

After dropping, the popup swings naturally using the `transform-origin` property.

```css
@keyframes swings;
```
